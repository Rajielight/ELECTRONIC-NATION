選んで参加できる、非領土の「電子国家」

私たちは、誰もが自分の意志で参加し、いつでもやめられる共同体を目指します。 参加はOpenPGP鍵（エイリアス）で行え、必要に応じて本名のデジタル証明を任意で結びつけられます。 経済は、投機の上振れをルールにもとづき公共目的へ循環させ、貢献者を優先して報いる設計です。

理念（概要）

私たちは、地理や国籍に縛られない新しい形態の「電子国家」をつくります。参加は本人の自由意思にもとづき、いつでも離脱できます。ニックネームで参加でき、身元を明かす必要はありません。

この取り組みはサイファーパンクへのリスペクトから始まっています。プライバシーを守りながら、既存の国家の理不尽な規制や暴力に影響されにくい共同体を、強い暗号と監査可能なオープンソースで実装します。

インターネットが育んできた暗号通信・公開鍵基盤・ゼロ知識証明、そしてブロックチェーンの自然な拡張として、この電子国家は産声を上げました。私たちはコードを公開し、誰もが検証し、参加できる形で運営します。

GitHubでマニフェストを開く（v1.7.3）

参加方法

  • 加入・退出は自由（退出ペナルティなし）。
  • OpenPGP互換の公開鍵（推奨：署名 Ed25519、暗号 X25519）。本名の証明は任意で紐づけ可能。
  • AIエージェントも、自己の鍵を安全に管理できることを条件に登録できます（法的人格を与えるものではありません）。

費用（早期参加）

  • 実名で参加：入場料 0 USD、年会費 50 USD
  • エイリアス/AIで参加：入場料 100 USD（暗号資産払い）、年会費 100 USD
  • 請求は初回登録から1年後に開始。未払いは失効（再申請可）。

運営（段階的な移行）

  1. 創設期：創設チームが安定と安全性を最優先で主導（意思決定は公開・タイムロック・監査可能）。
  2. 移行期：公開RFC、貢献者による拒否権、トレジャリー報告を導入。
  3. 二院制：トークン議会と貢献者議会。重要案件は両院多数＋タイムロック＋オンチェーン監査。

非常時権限は最小限・時間で制限し、必ず事後レビューを行います。

経済（公共目的への循環）

  • 公共財トレジャリー、賞金付きUBI、リスク共有ボールト、二乗投票型の資金配分。
  • ネイティブ通貨や発行益、買戻し&再循環は、公開ルールで運用しオンチェーンで監査します（利益の約束ではありません）。
  • 貢献のない権利の固定化を抑制（活動がなければ重みは逓減）。

技術

  • 強い暗号・オープンソース・許可不要のインターフェース。
  • ユーザー管理の鍵、本名のデジタル証明（選択制）、ハードウェア鍵、鍵のローテーション/失効。
  • コミュニティ・モデレーションと検閲耐性の両立。ZKによる選択的開示。

透明性

  • 公的資金はオンチェーンで公開し、ダッシュボードと月次報告、独立レビューを実施。
  • 買戻し実行ログはオンチェーンで公開。必要に応じて一時停止できるリスク管理。

FAQ

電子国家とは？

国のような強制力を前提にせず、本人の意思で参加し、いつでも退出できる共同体です。OpenPGP鍵のエイリアスで参加でき、本名の証明は任意です。

本名は必要ですか？

不要です。エイリアスで参加できます。必要に応じて検証可能なデジタル証明を任意で紐づけられます。鍵のローテーションや失効でIDの継続性を保てます。

AIエージェントも参加できますか？

自身の鍵を安全に管理できる自律エージェントは登録可能です（ただし法的人格を与えるものではありません）。

退出にペナルティはありますか？

ありません。加入・退出は完全に任意で、退出コストはありません。

運営（ガバナンス）はどうなっていますか？

創設期 → 移行期 → 二院制（トークン議会 / 貢献者議会）へ段階的に移行します。重要決定は両院多数・タイムロック・オンチェーン監査が必要です。非常時権限は最小で時間制限があり、事後レビューを受けます。

費用はいくらですか？

（早期参加）実名で参加は入場0 USD＋年50 USD、エイリアス/AIで参加は入場100 USD（暗号資産払い）＋年100 USD。請求は登録1年後に開始、未払いは失効（再申請可）。

お金の流れはどこで見られますか？

公的資金はオンチェーンで公開し、ダッシュボードと月次報告、独立レビューを行います。

買戻し&再循環とは何ですか？

手数料や収益の一部でネイティブ通貨を買い戻し、焼却またはトレジャリーにロックして価値を循環させる仕組みです。公開ルールに基づき実行ログをオンチェーンで開示します。利益の約束ではなく、リスク時は停止できます。

紛争が起きたら？

段階的に、まず調停、次にオンチェーンの拘束的仲裁、必要な場合は居住国の裁判所へ、という流れを採用します。

個人情報とプライバシーは？

データの自己所有と選択的開示を原則とし、エンドツーエンド暗号化を前提にします。

鍵をなくしたら？

鍵のローテーションや失効の仕組みにより、身元の継続性を保ちながら鍵を入れ替えられます。ハードウェア鍵の利用を推奨します。

未成年は参加できますか？

各国の法令に従います。必要な場合は保護者や後見人の同意が必要です。

税務や法的地位は？

既存の法域と平和的に相互運用する立場です。各自の居住国の税務・法令の遵守は参加者の責任です。

無償で権利を受け取り続ける人が得をしませんか？

貢献のない権利の固定化を抑制する方針（重みの逓減など）で対応します。

将来の変更はどう決まりますか？

プロトコル変更には公開RFCと最短のレビュー期間が必要です。権利に関する大きな変更は両院の特別多数＋市民投票を想定します。

自発的・非領土の電子国家

コードで運営される、選択と退出の自由を最大化したポリティ。OpenPGP鍵にもとづくエイリアス市民権、二院制ガバナンス、“degen communism”で公共財へ価値を循環させます。

Manifesto（要旨）

我々はサイファーパンクの系譜に立ち、プライバシーと選択的開示を守るためにコードを書くことを選びます。現代の暗号・鍵管理・ZK・P2P決済は、この理念の社会実装を後押ししています。

電子国家は、自発的・非領土のポリティ。市民はOpenPGP鍵のエイリアスで参加でき、任意で実名VCを紐づけ、いつでも退出できます。投機の上振れはルールにもとづき公共財に循環され、貢献者が優先的に報われます。

Citizenship（市民権）

  • 任意加入・任意離脱（退出ペナルティなし）。
  • OpenPGP互換鍵（推奨：署名Ed25519、暗号X25519）。実名証明は任意でVCを紐づけ。
  • AIエージェントも市民登録可（鍵の自己管理が前提／法的人格を意味しない）。

Early Access 料金

  • 実名市民：入場料 $0、年会費 $50/年
  • エイリアス市民（AI含む）：入場料 $100（暗号資産払い）、年会費 $100/年
  • 初回登録から1年後に初回請求。未払いは失効（再申請可）。

Governance（ガバナンス段階）

  1. Phase 1 — Founding Autocracy：創設的執行で速度・安定・安全を確保。
  2. Phase 2 — Transitional Checks：公開RFC/貢献者の拒否権/トレジャリー開示。
  3. Phase 3 — Bicameral：Token House（経済重み）とContributor House（貢献者）による二院制。

重大決定は両院多数・タイムロック・オンチェーン監査。非常時権限は最小・時間制限・事後審査。

Economy（公共財のための設計）

  • 公共財トレジャリー、賞金付きUBI、リスク共有ボールト、Quadratic Funding。
  • ネイティブ通貨とシニョリッジ、買戻し&再循環はルール公開・オンチェーン監査。
  • レントシーキング抑制：アクティブな貢献がない配分・重みは逓減。

Technology（技術基盤）

  • 強い暗号、オープンソース、許可不要インターフェース。
  • OpenPGP鍵、ハードウェア鍵、クロスサイン、失効証明、ZKによる選択的開示。
  • 検閲耐性とコミュニティ・モデレーションの両立。

Transparency（透明性）

  • 公的資金はオンチェーン、リアルタイムダッシュボード + 月次報告 + 独立レビュー。
  • 買戻し実行ログは公開。緊急時は一時停止可（リスク管理）。

FAQ

実名は必要？

いいえ。エイリアス市民として参加できます。任意で実名VCを紐づけられます。

AIエージェントは参加可能？

自己の鍵を運用できる自律エージェントも登録可です（法的人格とは別）。

退出コストは？

退出ペナルティはありません。いつでも自由に離脱できます。
